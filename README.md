# 🌊 Sonar Data Prediction (Rock vs Mine)

## 📌 Overview
This project is a Machine Learning-based system that predicts whether an object is a rock or a mine using sonar signal data.

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 📊 Dataset Information
The dataset contains sonar signals bounced off objects underwater. Each instance has 60 numerical features representing energy values of sonar waves.

## 📊 How it Works
1. Load and preprocess sonar dataset
2. Split data into training and testing sets
3. Train model using classification algorithms
4. Predict whether the object is Rock or Mine

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Input custom sonar values to test prediction

## 📈 Output
- Rock 🪨
- Mine 💣

## 📊 Model Performance
- Accuracy: 83%

## 📂 Project Structure
- sonar_prediction.ipynb → Main model
- requirements.txt → Libraries used
- Screenshot.png → Output image
